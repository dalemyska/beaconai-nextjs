## Content Format Template

Each blog post should follow this structure:

### Opening (150-200 words)
- Hook with a problem/pain point
- Establish credibility (MIT, Fortune 50 experience)
- Promise of what they'll learn
- Include primary keyword in first 100 words

### Body (1000-1500 words)
- H2 sections with secondary keywords
- Actionable frameworks/checklists
- Real examples from experience
- Data/statistics when available
- Industry-specific applications

### Closing (100-150 words)
- Summary of key takeaways
- Strong CTA (Assessment, Consultation, Download)
- Next article teaser

### Technical SEO
- **Word count:** 1200-1800 words
- **Headers:** H1 (title), H2 (main sections), H3 (subsections)
- **Images:** 2-3 custom images with descriptive alt text
- **Internal links:** 3-5 to other blog posts/pages
- **External links:** 2-3 to authoritative sources
- **Meta description:** 150-160 chars, include CTA
- **URL slug:** keyword-rich, max 60 chars

## Content Quality Standards

### Writing Guidelines
- **Tone:** Professional yet approachable, like a trusted advisor
- **Perspective:** First-person when sharing experience, second-person for advice
- **Avoid:** Jargon, buzzwords, AI hype, generic advice
- **Include:** Specific examples, frameworks, checklists, real numbers

### SEO Checklist
- [ ] Primary keyword in title, H1, first paragraph, URL
- [ ] Secondary keywords in H2s and naturally throughout
- [ ] Meta description optimized (150-160 chars)
- [ ] Alt text on all images
- [ ] Internal links to related content
- [ ] External links to authoritative sources
- [ ] Schema markup (Article, FAQ if applicable)
- [ ] Mobile-friendly formatting
- [ ] Page speed <3s load time

### Content Review Process
1. **Draft:** Write following template
2. **SEO Review:** Check keyword placement, meta tags
3. **Fact Check:** Verify all claims and statistics
4. **Edit:** Grammar, clarity, flow
5. **Visual:** Add images, charts, formatting
6. **Preview:** Test on mobile and desktop
7. **Publish:** Add to blog, schedule social
8. **Promote:** Share across all channels
